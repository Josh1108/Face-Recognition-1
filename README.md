As the name suggests, FIRE is a simple tool to take and store attendance in classroom settings. It uses state-of-the-art Deep Learning models to identify and mark attendance against a stored database. This is useful in virtual settings, and verifies identification of examinee.

## Main Features

1. Real-Time Attendance
2. Dashboard containing Attendance Metrics for Teacher and Admins
3. Attendance using a phone application
4. ERP and database UI to track attendance & add it automatically
