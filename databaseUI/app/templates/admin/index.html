How To Use

  • Create a new Database from the option available in the NavBar.
  • Make sure to Train on your Database before using it with the application.
  • To view your previously created databases go to 'Select Database' from NavBar.
  • Single entries can be Added/Deleted from inside the 'Select Database' option.
  • After training of the Database, It is ready to use with the application.
  • To fetch the attendance after the whole procedure just select the corresponding database from 'Select Database' and entires will be marked as present or absent.
